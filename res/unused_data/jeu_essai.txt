Jeu d’essai 1 : Partie Standard
   1-	Cliquer sur le bouton «partie Standard» pour commencer une partie

   2-	Plusieurs jeux d’essai s’offrent au joueur

   2.1-	Le joueur peut choisir de frapper une balle (voir jeu d’essai 3)

   2.2-	Le joueur peut choisir de voir le menu pause (voir jeu d’essai6)


Jeu d’essai 2 : Mode Libre
   1-	Cliquer sur le bouton «Mode Libre» pour commencer une partie

   2-	Plusieurs jeux d’essai s’offrent au joueur

   2.1-	Le joueur peut choisir de frapper une balle (voir jeu d’essai 3)

   2.2-	Le joueur peut choisir de voir le menu pause (voir jeu d’essai6)

   2.3-	Le joueur peut choisir de modifier la vitesse initiale de la balle (voir jeu d’essai 4)

   2.4-	Le joueur peut choisir de choisir le type de balle ( voir jeu d’essai 5)


jeu d’essai 3 : Frapper la balle
   1-	Cliquer sur le bouton «Lancer»

   2-	Attendre un délai de 1 seconde pour laisser du temps au joueur de se préparer pour frapper la prochaine balle

   3-	La balle est projetée à partir du côté droit de l’écran avec une vitesse initiale (voir jeu d’essai 4). Elle sera projetée vers la droite et en haut. Le nombre de balle lancé sera incrémenté de 1

   4-	Utiliser la souris pour frapper la balle. Plus la vitesse de la batte est grande, plus la balle sera projetée loin. La caméra continuera de suivre la balle à partir du moment de frappe

   5-	Lorsque la balle touche le sol pour la première fois, le score est entré en fonction de la distance parcourue par la balle

   6-	Attendre 10 secondes depuis le début  du jeu d’essai 3 pour que la balle, la caméra et les composants du jeu se réinitialisent


jeu d’essai 4 : Changer la vitesse initiale de la balle
   1-	Utiliser la souris pour bouger le curseur rond bleu en enfonçant la souris et le déplacer

   2-	Le coin supérieur gauche représente la vitesse maximale en x et en y et le coin inférieur droit représente la vitesse minimale en x et en y

   3-	Lorsque le curseur est déplacé, les valeurs de vitesse en x et en y affichées dans les encadrés sont constamment ajustées

   4-	Relâcher la souris lorsque la vitesse initiale de la balle semble satisfaisante


jeu d’essai 5 : Activer/désactiver les types de balle que le générateur aléatoire peut choisir
   1-	Cliquer sur le bouton sans texte

   2-	Tous les boutons du jeu disparaissent et un panneau noir apparait

   3-	Choisir d’activer ou de désactiver les types de balle en cliquant sur les boutons associés au type de balle

   4-	Tous les types de balle sont activés par défaut

   5-	Cliquer sur le bouton avec un «X» pour retourner au jeu


jeu d’essai 6 : menu de pause
   1-	Cliquer sur le bouton pause

   2-	Le menu de pause apparait avec des boutons d’option

   3-	Cliquer sur «Reprendre» pour retourner à la partie

   4-	Cliquer sur le bouton «Aide» pour accéder au menu d’aide

   5-	Cliquer sur «Abandonner la partie» pour quitter le jeu et retourner au menu principal 


Le but de cette application pour la version Alpha est de montrer les mouvements de MUA de la balle projetée dans les airs. Il est possible d'observer l'effet de la gravité sur la balle. La balle est capable de détecter la collision avec le sol et les murs dans le monde de jeu. La vitesse de la balle qui rebondit dépend du matériel de la platfrome dans laquelle la balle collisionne.
